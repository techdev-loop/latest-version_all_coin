Polymarket Hedge Bot — Browser Simulator

Mirrors the paper backtest loop in src/scripts/simulateBacktest.ts (synthetic order books + hedgeStrategy.decide + maker fills). Open DevTools → Console (F12) for full logs. Long runs yield to the event loop in chunks so the tab stays responsive.

Optional — replay saved quotes (overrides synthetic books). Paste a JSON array of ticks; each element may be { "yesAsk", "noAsk" } or include bids: { "yesBid", "noBid", "yesAsk", "noAsk" }. One object = one tick; shorter arrays repeat last tick.

In the console you can call PolymarketSim.runSynthetic() or inspect PolymarketSim.lastSummary after a run.
